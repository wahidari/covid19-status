# Covid 19 Status

![Covid 19 Demo](/screenshot.png)

### Setup
 - didalam directory jalankan perintah berikut:
 - `npm install`
 - Untuk proses develop jalankan perintah berikut:
 - `npm run start-dev`
 - Untuk proses build jalankan perintah berikut:
 - `npm run build`
